> 前些日子开始使用Python，写了一段爬虫程序作为练习。然后诡异的是无论如何都只能抓取到一个链接的数据，页面分析器直接无法运行。今天晚上重新打开，一个字母一个标点的去死扣后还是无法发现问题所在。

* 报错内容："AttributeError("'NoneType' object has no attribute 'find'",)"


* 因为代码中使用了BeautifulSoup的find方法，一直卡在这一句上。


{% highlight Python linenos %}

def _get_new_data(self, page_url, soup):
        res_data = {}
        res_data['url'] = page_url
        title_node=soup.find('dd',class_="lemmaWgt-lemmaTitle-title").find("h1")
        res_data['title'] = title_node.get_text()
        summary_node = soup.find("div",class_="lemma-summary")
        res_data['summary'] = summary_node.get_text()        
        return res_data
        
{% endhighlight %}

* 根据报错原因可以分析出soup应该是NoneType，但是诡异的是它真的不是·······（头顶一排乌鸦飘过）！
