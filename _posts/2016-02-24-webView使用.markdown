##UIWebView加载本地HTML文件

> 需求，声明版权页面不再要跟服务端交互而直接取本地页面来显示。考虑过直接显示PDF文件，效果很差，rtf文件，乱码几率高。最后决定把.doc文件转换为.html文件直接加载到webView上。并且附在一个ViewController上实现Navigation的push控制界面跳转。

* 代码如下：


{% highlight Objective-C %}
- (void)PushToWebViewWithFileName:(NSString *)fileName {
    UIWebView *webView = [[UIWebView alloc] init];
    NSString *path = [[NSBundle mainBundle] pathForResource:fileName ofType:@"html"];
    NSURLRequest *request = [NSURLRequest requestWithURL:[NSURL fileURLWithPath:path]];
    webView.scalesPageToFit = NO;
    // NSString *htmlString = [NSString stringWithContentsOfFile:path encoding:NSUTF8StringEncoding error:nil];
    //[webView loadHTMLString:htmlString baseURL:[NSURL URLWithString:path]];
    [webView loadRequest:request];
    UIViewController *viewController = [[UIViewController alloc] init];
    viewController.view.frame = [[UIScreen mainScreen] bounds];
    viewController.view = webView;
    [self.navigationController pushViewController:viewController animated:YES];
}
{% endhighlight %}
